使用本程序可使Windows电脑键盘及鼠标只在授权通过后正常使用。

View on GitHub Download .zip Download .tar.gz

欢迎使用键盘拦截程序

程序功能及使用说明

1.程序正常运行后可拦截所有Windows键盘按键（Ctrl + Alt + Delete除外）及鼠标按键；
2.程序可选择使用Windows文件共享授权或使用MySql数据库授权;
3.程序正常运行需要正确配置数据库信息或本地授权文件及日志文件存储路径及各文件名、共享服务器端IP地址，用户名及密码；
4.程序内相关参数修改请参考文件ParametersDefine.cs中参数定义;

程序开发运行环境

1.开发： Windows7 x64 ，VS2012 ，.Net 4.5；
2.运行： .Net 4.5
3.数据库： MySql v5.7

程序其他说明

注意：
主程序并无界面显示，运行后直接开始拦截；
